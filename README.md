# Protocol Clients and Servers

A comprehensive collection of client and server implementations for MQTT, Socket, and OPC UA protocols, designed for testing and demonstration purposes. This repository provides implementations in Python, C#, and JavaScript.

## 🚀 Features

This repository contains implementations for three major communication protocols:

### MQTT
- MQTT client implementation
- Support for publish/subscribe messaging patterns
- Configurable connection settings

### Socket
- TCP/IP Socket client and server implementations
- Bi-directional communication support
- Customizable message handling

### OPC UA
- OPC UA client and server implementations
- Industrial automation protocol support
- Secure communication channels

## 🛠️ Technology Stack
